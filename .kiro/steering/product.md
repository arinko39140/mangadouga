# Product Overview

Front-end prototype for a personal "oshi" (favorites) profile, showcasing curated lists of creators/works with a warm, hand-crafted visual style.

## Current Direction

推しができる漫画動画サイトを最終目標とし、Supabaseを用いた動的サイトとして構築する。

## Core Capabilities

- Profile-style landing page with multiple curated sections
- Dedicated list pages for "oshi" items and works
- Public/private visibility toggles persisted in the browser
- Static content authored as Markdown with embedded HTML
- 推しの漫画・動画の紹介ページと一覧表示
- 作品情報やクリエイターの見せ方に重点を置いたUI

## Target Use Cases

- Personal profile for sharing favorite creators/works
- Lightweight static pages for design exploration and iteration
- Rapid UI prototyping without a backend

## Value Proposition

Delivers a fast, static front-end experience with expressive styling and simple client-side behavior, keeping content editable and deployable without server infrastructure.
